NetHelper Class

Class instance accessor: Net

.WebService

Set a web service function to be called.

.WebService(  serviceurl )Returns: NetHelper

serviceurl
Type: string
External Web Service function url.
Example snippet:
		var iplocation = Net.WebService("http://freegeoip.net/json/").GetData(); 
		Program.Notify("IP to Location", iplocation.city);

.AddHeader

Adds a custom header to the web service call.

.AddHeader( header )Returns: NetHelper

header
Type: string
String containing the http header to send.

.Call

Call the web service url specified with .WebService (see above).

.Call()Returns: void

Note: the execution will block until the request is completed. No return value is provided.

.GetData

Call the web service url specified with .WebService and returns the server response.

.GetData()Returns: dynamic

The returned value can be a simple string or an object containing all fields mapped from the JSON/XML response. (see .WebService example)

.GetBytes

Call the web service url specified with .WebService and returns a raw byte array containing the server response.

.GetBytes()Returns: byte[]

This function is meant to be used with .AddAttachment, for sending e-mail with images or any other binary data.

.Post

Use POST method when calling the webservice.

.Post( postdata )Returns: NetHelper

postdata
Type: string
String containing post data fields and values in the form field1=value1&filed2=value2&...&fieldn=valuen.

.Put

Use PUT method when calling the webservice.

.Put( putdata )Returns: NetHelper

putdata
Type: string
String containing post data fields and values in the form field1=value1&filed2=value2&...&fieldn=valuen.

.Ping

Ping a remote host.

.Ping( host_address )Returns: bool

ip_address
Type: string
IP or DNS address of the remote host to ping.

.WithCredentials

Set username and password for WebService or MailService operations.

.WithCredentials( username, password )Returns: NetHelper

username
Type: string
User name.
password
Type: string
User password.

.MailService

Set the SMTP server address used for sending e-mail with .SendMessage.

.MailService( smtp_serveraddress )Returns: NetHelper

If .MailService is not called when using .SendMessage, default system e-mail account settings will be used.

.SendMessage

Send an E-Mail.

.SendMessage( recipients, subject, messagetext )Returns: bool

recipients
Type: string
Comma separated list of recipients e-mail.
subject
Type: string
Subject of the e-mail.
messagetext
Type: string
Text of the e-mail.
The return value will be true if the e-mail has been succesfully delivered, false otherwise.

.AddAttachment

Add a file attachment to the e-mail.

.AddAttachment( filename, rawdata )Returns: NetHelper

filename
Type: string
The file name with extension (eg. image1.jpg).
rawdata
Type: byte[]
File raw data byte array.
Called before .SendMessage, can be used to attach one or more files.

.SignalModuleEvent

Route a module parameter change event to a remote HG server.

.SignalModuleEvent( hg_address, module, parameter )Returns: NetHelper

hg_address
Type: string
Address of the remote HG server.
module
Type: ModuleHelper
The module instance.
parameter
Type: ModuleParameter
The parameter to signal.
Note that authentication is currently unsupported.
